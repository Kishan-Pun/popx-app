# 🚀 PopX App – Frontend Assignment

A pixel-perfect mobile-first web application built using **React + Vite + Tailwind CSS**, implementing authentication flows and clean UI components.

---

## 📌 Live Demo

🔗 https://popx-app-orcin.vercel.app/

---

## 📌 GitHub Repository

🔗 https://github.com/Kishan-Pun/popx-app

---

## ✨ Features

* 📱 **Mobile-first UI** (centered mobile layout)
* 🎯 **Pixel-perfect design implementation**
* 🔐 **Authentication flow**

  * Register → Login → Account
* ⚡ **Smooth navigation using React Router**
* 🧩 **Reusable components**

  * Button
  * Input (floating label style)
* ✅ **Form validation (basic)**
* 🎨 **Clean and modern UI with Tailwind CSS**
* 📦 **Built with TypeScript for type safety**

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Language:** TypeScript
* **Styling:** Tailwind CSS (v4)
* **Routing:** React Router DOM
* **Icons:** Lucide React

## ⚙️ Setup & Installation

Clone the repository:

```bash
git clone https://github.com/Kishan-Pun/popx-app.git
cd popx-app
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🚀 Deployment

The project can be deployed easily using:
